Lo script di Matlab presente in questo folder permette di analizzare la postura dell'anca rispetto ad un sistema di riferimento inerziale, denominato WORLD FRAME. In questo caso è stato considerato il piede puntiforme. Inoltre, verrà analizzata anche la traiettoria generata dal piede della gamba in oscillazione (gamba in "swing") rispetto al sistema di riferimento inerziale WF.
