Lo script di Matlab presente in questo folder permette di analizzare la postura dell'anca durante un passo del walking robot rispetto ad un sistema di riferimento inerziale, denominato World Frame (con l'origine coincidente con la punta del piede).
La simulazione salvata nel file .avi permette quindi di analizzare la traiettoria dell'anca durante un passo ed inoltre permette di confrontare la traiettoria della punta del piede rispetto alla traiettoria della caviglia del piede della gamba in swing.
