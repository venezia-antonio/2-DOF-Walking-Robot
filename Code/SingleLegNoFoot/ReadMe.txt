Lo script di Matlab in questo folder permette di determinare posizione e orientamento di tutti i link che costituiscono la singola gamba del walking robot, una volta note le variabili di giunto attuate e non attuate. L'output di questa simulazione è la posa dell'end effector al variare dei valori assunti dalle variabili di giunto. 
In aggiunta, è presente un file .avi in cui si mostra la traiettoria generata dall'end effector, oltre che la figura di Hoeken.
