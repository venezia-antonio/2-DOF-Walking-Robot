Lo script di Matlab presente in questo folder permette di determinare la traiettoria della punta del piede della gamba in swing rispetto ad un sistema di riferimento inerziale posizionato sulla punta del piede della gamba in appoggio. Inoltre, è possibile analizzare la postura dell'anca durante il passo effettuato dal walking robot.
É opportuno osservare che in questo caso è stato considerato il doppio parallelogramma articolato per ogni gamba del walking robot.
La simulazione è salvata come file .avi, denominato "BodyPostureStabFoot.avi".
