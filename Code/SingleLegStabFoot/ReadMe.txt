Lo script di Matlab presente in questo folder permette di determinare la posa dell'organo terminale rispetto al sistema di riferimento base in 0 quando nel meccanismo della singola gamba viene considerato anche il doppio parallelogramma articolato. 

Tutti i dati relativi alle variabili di giunto delle due gambe rispetto al sistema di riferimento base 0 sono salvate nel file denominato "TwoLegStabilizerData.mat": caricando questo file non sarà necessario compilare le righe di codice precedenti. 
Il movie mostra il gait pattern della punta del piede (end effector) di una sola gamba ed è denominato "SingleLegStabFoot.avi".
