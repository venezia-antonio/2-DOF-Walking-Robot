Lo script di Matlab in questo folder permette di determinare posizione e orientamento di tutti i link che costituiscono la singola gamba del walking robot, una volta note le variabili di giunto attuate e non attuate. 
Nel meccanismo della singola gamba, viene considerato un piede di lunghezza fissata rigidamente connesso alla tibia con un angolo tra tibia e piede di 90Â°.
In questa simulazione, in particolare nel file .avi, viene confrontata la traiettoria generata dalla punta del piede con la traiettoria generata dal giunto di caviglia.
